# First Steps

このリポジトリは [t3turbo](http://github.com/t3-oss/create-t3-turbo) を使用して開始しました。その時の記録です。

[T3 turbo](https://github.com/t3-oss/create-t3-turbo)を使う方の参考においておきます。このリポジトリを使う方は、以下の手順はすでに終わっていますので、自分に必要なところだけ再設定する際の参考にしてください。

## セットアップ

1. リポジトリをクローンしました。

```sh
   npx create-turbo@latest -m pnpm -e http://github.com/t3-oss/create-t3-turbo
   cd t3turbo-template
   code .
```

リポジトリを取得後、`create-t3-turbo` が `pnpm install` を実行しました。

2. パッケージprefixを `@acme` から `@myprefix`(このテンプレートでは`@aromarious`)に変更しました。（vscodeの置換を使用）
   一度ワークスペースを閉じてファイルを保存してから、commitしました。

3. 1.でインストールされた依存関係を一度削除し、再インストールを実施しまsたい。

```sh
   pnpm clean:workspaces
   pnpm install
```

4. 次にリポジトリ内に用意してある開発ツールのセットアップをします。
   まず `prettier` の設定を行います。

   1. [index.js](./tooling/prettier/index.js) を編集します。特にセミコロンやクオートなどコード全体に影響のあるものを設定します。
   2. 動作確認をします。`pnpm format ファイル名`, `pnpm format:fix ファイル名`で確認できます。
   3. `pnpm format:fix`で全体のフォーマットを行います。
   4. 変更をコミットします。

パッケージprefixの変更と`prettier`の設定はプロジェクト全体に影響するため、最初に行うことをお勧めします。

`t3-turbo`の最低限の設定はこれで完了です。
