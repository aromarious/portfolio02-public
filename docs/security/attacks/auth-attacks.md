# 認証・認可攻撃の詳細

## パスワード総当たり

**攻撃手法**: ログイン認証に対する連続的なパスワード試行攻撃
**特徴**:

- 一般的なパスワード辞書の使用
- 分散攻撃による検出回避
- 自動化ツールの活用

**影響**:

- アカウントの乗っ取り
- 認証システムの負荷
- ユーザーデータの漏洩

**検出方法**:

- 連続する認証失敗
- 同一IPからの大量ログイン試行
- 短時間での多数アカウント試行

## トークン推測

**攻撃手法**: APIトークンやセッションIDの推測・総当たり攻撃
**特徴**:

- 弱いトークン生成アルゴリズムの悪用
- トークンパターンの解析
- 時間的な推測攻撃

**影響**:

- 他ユーザーのセッション乗っ取り
- APIの不正利用
- システムへの不正アクセス

**検出方法**:

- 無効なトークンでの連続アクセス
- トークン生成パターンの分析
- 異常なトークン使用パターン

## セッション固定

**攻撃手法**: セッションIDを固定化して他ユーザーのセッションを乗っ取る攻撃
**特徴**:

- ログイン前のセッションIDの固定
- クロスサイトスクリプティングとの組み合わせ
- セッション管理の脆弱性悪用

**影響**:

- ユーザーのなりすまし
- 認証状態の乗っ取り
- 個人情報の漏洩

**検出方法**:

- セッションIDの再生成失敗
- 異常なセッション継続時間
- 複数IPでの同一セッション使用

## セッションハイジャック

**攻撃手法**: 有効なセッションIDを盗取してユーザーのセッションを乗っ取る攻撃
**特徴**:

- ネットワーク盗聴による取得
- XSS攻撃との組み合わせ
- 中間者攻撃の活用

**影響**:

- 完全なアカウント乗っ取り
- 機密データへのアクセス
- 不正な操作の実行

**検出方法**:

- 異常なセッション活動
- 地理的に離れた場所からの同時アクセス
- セッション情報の急激な変化
